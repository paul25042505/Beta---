版本修改紀錄

v1.5.0 2026-04-14　Firebase 全面整合 最新

🔥 Firebase 整合：資料全面遷移至 Firestore，登入驗證、成員、物品、簽到簽退、協勤紀錄均從 Firebase 讀寫
📊 報表資料來源：簽到簽退、服勤表、協勤紀錄表改從 Firestore 抓取
⚙️ 管理員後台：帳號管理（角色/分隊/證照）、簽到簽退/協勤紀錄查詢、物品管理、意見回饋、公告管理
🏢 分隊別架構：分隊管理、帳號與成員依分隊篩選，報表/統計依分隊匯出
💬 意見回饋：可選分隊、匿名或具名，管理員/承辦人可查看
👤 角色權限：系統管理員、分隊承辦人、一般成員，報表僅管理員/承辦人可用
📢 公告系統：首頁公告橫幅與緊急公告彈窗

v1.4.05 2026-03-26　功能補強

小修
✨ 新功能
  • 簽到完成後，若有待領物品尚未領取，自動彈出提示視窗顯示物品清單，可直接前往領取
  • 待領物品提示依據白名單 B 欄義消姓名比對，每人只看到自己的待領物品
  • 服務項目新增「靜脈注射（IV）」
🎨 UI 調整
  • 簽退完成畫面改為「查看簽到紀錄」按鈕（綠色框線樣式）
  • 簽到頁面類別選項與上下欄位間距統一
🐛 Bug 修復
  • 協勤紀錄查詢移除「協勤類別」篩選欄位
  • 修正登入驗證錯誤（data 變數未定義）

v1.4.04 2026-03-25　類別邏輯調整

小修
✨ 新功能 / 調整
  • 簽到登錄新增「類別」選項（協勤／公差），於簽到時選擇，記錄至 Sheets B 欄
  • 協勤案件登錄移除「協勤類別」選項
  • 簽退查詢表格新增「類別」欄
  • 協勤紀錄表改為自動分頁，每頁重複標題列
  • 忘記簽退可無限期補簽，標示橘色「待補簽」
  • 補簽退時日期使用簽到當天日期
🐛 Bug 修復

  • 確保「協勤紀錄」工作表有標題列，欄位不再錯位

v1.4.01 2026-03-22　物品領用

中版本更新
✨ 新功能
